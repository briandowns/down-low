# down-low

down-low is a secure messaging system written in Go with the ability to send AES and RSA encrypted messages to a number of different endpoints.

## Currently

- Gmail

## Coming Soon...

- Twitter
- Facebook

## Warning

This application is _not yet_ functional.  Work in progress.
